# RoamingSnail

RoamingSnail is an OpenGL/C++ project based on the previous 8-Ball-Pool engine.

The goal of the project is to implement a 3D roaming snail simulation/game with:

- procedural heightmap terrain with hills and valleys
- a snail player that sticks to the terrain
- retractable shell mode
- shell rolling physics
- friction and spin
- textured terrain with random UV variation
- lighting and skybox
- vegetation interactions
- fun rolling, bouncing, and slingshot mechanics

## Current Status

Initial project setup based on the previous 8-Ball-Pool repository.
